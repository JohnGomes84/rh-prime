# Fase 5 — Fechamento de Produto (Aceite)

## Objetivo

Formalizar o aceite funcional, técnico e operacional para liberar o FinHub com rastreabilidade.

## 1) Aceite funcional (negócio)

- [ ] Fluxo de planejamento completo validado (criar -> alocar -> validar -> fechar).
- [ ] Fluxo de pagamento sem duplicidade validado.
- [ ] Fluxo de alteração PIX com aprovação validado.
- [ ] Fluxo de presença (check-in/check-out/falta) validado.

## 2) Aceite técnico

- [ ] `pnpm run check` verde.
- [ ] `pnpm run test` verde ou com suíte de integração segregada e documentada.
- [ ] Migration aplicada em homologação e conferida.
- [ ] Logs com correlation id e trilha de auditoria verificados.

## 3) Aceite operacional

- [ ] Backup executado antes de go-live.
- [ ] Restore testado em ambiente de homologação.
- [ ] Health/readiness monitorados.
- [ ] Checklist de deploy e rollback assinado.

## 4) Aprovação CLT/LGPD (Brasil)

- [ ] RH/DP validou regras de jornada e presença.
- [ ] Financeiro/Contábil validou fechamento e lotes.
- [ ] Jurídico/Compliance validou retenção e auditoria LGPD.
